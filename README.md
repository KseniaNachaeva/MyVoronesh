# MyVoronesh

**MyVoronesh** — это мобильное приложение для прохождения интерактивных городских квестов в Воронеже. Пользователи могут исследовать город, находить локации, выполнять задания и отслеживать свой прогресс.

##  Функциональность

*   **Авторизация:** Регистрация и вход в систему.
*   **Квесты:** Список доступных маршрутов по городу.
*   **Карта:** Интерактивная карта с точками квестов (Yandex Maps).
*   **Профиль:** Личный кабинет пользователя со статистикой.
*   **Фото-отчеты:** Возможность загружать фотографии пройденных мест.

##  Технологический стек

### Android (Клиент)
*   **Язык:** Kotlin
*   **UI:** Jetpack Compose (Material Design 3)
*   **Архитектура:** MVVM
*   **Сеть:** Retrofit 2 + OkHttp + Gson
*   **Карты:** Yandex Maps Mobile SDK
*   **Изображения:** Coil

### Backend (Сервер)
*   **Фреймворк:** Ktor
*   **Язык:** Kotlin
*   **База данных:** MySQL
*   **ORM:** Exposed
*   **Безопасность:** BCrypt (хеширование паролей)

##  Установка и запуск

### Предварительные требования
*   JDK 17+
*   Android Studio Ladybug или новее
*   MySQL Server

### Шаг 1: Настройка базы данных
1.  Установите и запустите MySQL.
2.  Создайте пустую базу данных с именем `MyVoronesh`.
3.  Импортируйте структуру и данные из дампа, который находится в папке `database/`:
    *   **Через phpMyAdmin:** Выберите базу `MyVoronesh` -> Импорт -> Выберите файл `database/init_db.sql`.
    *   **Через консоль:**
        ```bash
        mysql -u root -p MyVoronesh < database/init_db.sql
        ```

### Шаг 2: Запуск Backend (Сервер)
1.  Откройте папку `backend` в IntelliJ IDEA или терминале.
2.  Проверьте настройки подключения к БД в файле `backend/src/main/kotlin/config/DataBase.kt`. По умолчанию:
    *   User: `root`
    *   Password: `""` (пустой)
    *   Port: `3306`
3.  Запустите сервер:
    ```bash
    cd backend
    ./gradlew run
    ```
    Сервер запустится на порту `8080`.

### Шаг 3: Запуск Android (Приложение)
1.  Откройте папку `android` в Android Studio.
2.  Настройте адрес сервера в файле `android/app/src/main/java/com/example/myvoronesh/data/remote/ApiClient.kt`:
    *   Если используете **Эмулятор**: оставьте `http://10.0.2.2:8080/`
    *   Если используете **Реальное устройство**: укажите локальный IP вашего компьютера, например `http://192.168.1.5:8080/` (телефон и ПК должны быть в одной Wi-Fi сети).
3.  Синхронизируйте Gradle (`Sync Project with Gradle Files`).
4.  Запустите приложение на устройстве или эмуляторе.
